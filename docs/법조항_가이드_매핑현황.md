## 5. 법조항 vs KOSHA GUIDE 내용 성격 비교

### 5.1 근본적 차이

| 구분 | 법조항 (산안법 시행규칙) | KOSHA GUIDE |
|------|------------------------|-------------|
| **성격** | 의무 규정 ("~하여야 한다") | 실무 지침 ("~하는 방법") |
| **초점** | **무엇**을 해야 하는가 (What) | **어떻게** 해야 하는가 (How) |
| **예시** | "추락 방지 조치를 하여야 한다" | "추락방호망 설치 시 수직형은 XX간격으로..." |
| **법적 구속력** | 있음 (위반 시 처벌) | 없음 (권고 사항) |
| **범위** | 포괄적, 모든 산업 | 특정 작업/설비에 특화 |
| **갱신 주기** | 법률 개정 시 | 수시 (연도별 발행) |

### 5.2 내용 유형 매트릭스

```
                  절차/관리           안전/위험           설비/구조
              ┌───────────────┬───────────────┬───────────────┐
  법조항      │ "점검해야 한다" │ "방지해야 한다" │ "구조를 갖춰야" │
  (What)      │  68개 (33%)   │  63개 (30%)   │  77개 (37%)   │
              ├───────────────┼───────────────┼───────────────┤
  KOSHA GUIDE │ "측정 방법은"  │ "안전작업 절차" │ "설비 시방서"  │
  (How)       │  52개 (15%)   │  198개 (58%)  │  93개 (27%)   │
              └───────────────┴───────────────┴───────────────┘

  ※ 법조항은 절차/설비 비중이 높고
  ※ KOSHA GUIDE는 안전/위험 실무 비중이 압도적 (58%)
```

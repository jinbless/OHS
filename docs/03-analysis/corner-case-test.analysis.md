### 2-2. 강점 분석
- **복합 위험 100%**: 추락+감전, 화재+질식 등 복수 위험이 겹치는 시나리오에서도 정확하게 조문 특정
- **모호한 표현 100%**: 법률 용어 없이 일상 언어로 묘사해도 올바른 조문 매칭
- **드문 위험 100%**: 방사선, 석면, 벌목, 잠수 등 빈도 낮은 위험도 정확 매칭

### 2-3. 취약점 분석

#### ❌ 설비 규격 (2/6 = 33%) - 가장 취약
실패 조문:
- **제120조** (안전밸브): 보일러 안전밸브 설정압력 시나리오 → 매칭 실패
- **제174조** (이탈방지): 컨베이어 벨트 이탈 → 매칭 실패
- **제222조** (대피): 유해물질 대피 경로 → 매칭 실패
- **제261조** (건조설비 통풍): 건조설비 내부 폭발 → 매칭 실패

**원인 추정**: 특정 설비 규격 조항이 ChromaDB에 임베딩 품질이 낮거나, GPT hint 프롬프트에서 누락

#### ❌ 관리/행정 조항 (3/6 = 50%) - 취약
실패 조문:
- **제115조** (과부하 제한): 크레인 과부하 → 매칭 실패
- **제96조** (도구 목적외 사용): 공구 오용 → 매칭 실패

**원인 추정**: 관리적 성격의 조항은 키워드 매칭이 어렵고, 시나리오 → 법적 의무 연결이 약함

#### ⚠️ 기존 실패 재도전 (5/8 = 62%)

## 4. 개선 방향 제안

### 즉시 개선 가능 (Quick Win)

1. **설비 규격 조항 힌트 보강**
   - 제120조(안전밸브), 제174조(이탈방지), 제222조(대피), 제261조(건조설비)를
   - GPT hint 프롬프트 또는 ChromaDB 메타데이터에 키워드 추가

2. **관리 조항 키워드 매핑**
   - 제115조: "과부하", "정격하중", "하중제한"
   - 제96조: "도구 목적외", "공구 오용"
   - 제669조: "VDT", "영상단말기", "컴퓨터 작업"

3. **KOSHA W분류 시소러스 확장**
   - "MSDS", "물질안전보건자료" → W분류 우선 매핑
   - "한랭", "냉동", "저온 작업" → W분류 우선 매핑

### 중기 개선 (Medium Term)

4. **G분류 특수 가이드 별도 처리**
   - 동물원, 공연, 화약 등 특수 작업장 키워드 → 해당 G분류 가이드 직접 연결

5. **H분류 보건 가이드 세분화 검색**
   - 증상/질환명 → 특정 H분류 가이드 매핑 테이블 구축
   - 예: "피부염" → H-73, "구강" → H-211, "COPD" → H-95

6. **법령-KOSHA 교차 매핑 강화**
   - 법령 조문과 관련 KOSHA 가이드를 reg_guide_mapping 테이블에 추가

### 장기 개선 (Long Term)

7. **다단계 리랭킹 파이프라인**
   - 1차: ChromaDB 벡터 검색 (현재)
   - 2차: 키워드 기반 필터링 (추가)
   - 3차: LLM 리랭킹 (현재)

8. **실패 패턴 기반 자동 테스트 회귀**
   - 개선 후 코너케이스 자동 재실행하여 회귀 방지
